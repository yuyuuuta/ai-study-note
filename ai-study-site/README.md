# AI学習ノート（生成AIパスポート対策）

自分専用のAI学習サイトです。

## 使い方
1. このフォルダ内の `index.html` をブラウザで開くだけ
2. すべてオフラインで動作します（ネット接続不要）

## 構成
- index.html ... トップページ（章の一覧）
- chapter1.html ... 1章：AIの基礎と全体像
- chapter2.html ... 2章：生成AIのアーキテクチャ
- style.css ... デザイン
- script.js ... インタラクティブ動作
- assets/ ... キャラクター画像

## 機能
- カタカナ用語にマウスを乗せると補足が出ます（点線の下線がついた語）
- アコーディオン（クリックで開閉）
- タブ切り替え
- スクロールで要素がふわっと表示
- スマホ対応

## 編集のしかた
各HTMLファイルをテキストエディタで開けば、内容を追記・修正できます。
新しい章を追加したいときは、chapter2.html をコピーして chapter3.html を作り、
index.html とナビゲーションにリンクを足してください。
